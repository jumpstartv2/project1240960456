Thank you for using Jumpstart!

You can start creating apps by following the steps below:

  • Navigate to apps directory:

    $ cd apps/

  • Create your app:

    $ python ../manage.py startapp <app_name>

    or

    $ django-admin startapp <app_name>

  • Open common.py in your text editor

  • Add apps.<app_name> in INSTALLED APPS

    Note: It should follow this standard except for DjangoCMS

    # Django

    # Third-party Apps

    # Local apps

  • You can now start creating your models, views, admin, and etc.
